Задание сделал в виде WP плагина с использованием jQuery
https://github.com/dmitry-rozhkov/smartapp_test

Использование:
1) подключить плагин (название: "Custom Form" )
2) заполнить настройки > Admin panel > Настройки > Custom Form 
3) на страницу добавить шорткод [custom_form]

При включенном CF_DEBUG (custom_form.php) стр 9 отображаются технические сообщения:
1) данные email_to, subject, message  отправленного сообщения
2) запись в лог, OK или ошибка 
3) Добавление контакта в hubspot.com, OK или ошибка 

Лог файл создайтся в папке с плагином, название custom_form.log 

При создании hubspot.com Коннтакт добавляется с полями из формы  First Name, Last Name, E-mail

Данные из формы Subject и Message не используются согласно заданию.
